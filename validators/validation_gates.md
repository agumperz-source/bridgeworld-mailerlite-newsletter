# Validation Gates

## Hard fail

- schema invalid
- source bridge inconsistency
- impossible full deal/auction/play sequence
- ambiguous problem/solution pairing
- renderer uses HTML as semantic source truth outside REVISE_HTML parse-back
- section order broken
- article content appears after Stay Connected/footer
- production HTML contains legacy MailerLite suit/logo URLs
- production HTML exceeds hard size budget unless user explicitly authorizes

## Warning

- production HTML exceeds target size budget
- export is visually untested after Tier 2 optimization
- ambiguous but non-critical source metadata
- asset URL host is valid but not preferred short canonical host
