# Bridgeworld Mailerlite Newsletter Project

Repository workspace for the Bridgeworld Mailerlite newsletter project.

## Purpose

This project tracks newsletter planning, MailerLite implementation notes, campaign assets, and reusable operational documentation for Bridgeworld email publishing.

## Suggested Structure

- `assets/` - images, exports, and other newsletter media
- `docs/` - setup notes, process documentation, and campaign planning
- `templates/` - reusable copy, layouts, and campaign patterns

## Getting Started

1. Add project files to the folders above as the newsletter workflow takes shape.
2. Keep private keys, MailerLite API tokens, subscriber exports, and contact lists out of this repository.
3. Use issues or project notes to track newsletter tasks, campaign launches, and integration work.
